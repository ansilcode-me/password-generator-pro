# 🔐 Password Generator Pro

A simple GUI-based password generator with strength checker.

---

## 🚀 Features
- Generate strong passwords 🔑
- Password strength indicator (Weak / Medium / Strong)
- Copy password to clipboard 📋

---

## 🖥️ Requirements
- Python 3
